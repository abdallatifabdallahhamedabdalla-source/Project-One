# Leon | Creative Agency Template

A responsive landing page built using **HTML5** and **CSS3**.

## 📌 About

Leon is a modern and clean landing page template created as a frontend practice project. The website is fully responsive and built without any CSS frameworks.

## 🚀 Features

* Responsive Design
* Flexbox Layout
* CSS Grid Layout
* CSS Variables
* Smooth Scrolling
* Clean and Organized Code
* Modern UI

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Font Awesome
* Google Fonts
